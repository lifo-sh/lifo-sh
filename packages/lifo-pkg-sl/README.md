# lifo-pkg-sl

Steam locomotive (`sl`) command for [Lifo](https://github.com/lifo-sh/lifo) â€” based on the classic [sl](https://github.com/mtoyoda/sl) by Toyoda Masashi.

Run it when you mistype `ls`.

## Install

```bash
lifo install sl
```

## Usage

```bash
sl
```

A D51 locomotive rolls across your terminal. Press `Ctrl+C` to skip it.
